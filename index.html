The Core Concept

What is Agentic AI?

Agentic AI goes beyond generation — it perceives, plans, acts and iterates autonomously to achieve a defined goal, using tools and memory across multi-step tasks.

Gen AI vs Agentic AI
Gen AI: Prompt in → Answer out (one shot)
Agentic AI: Goal in → Plans → Acts → Observes → Retries → Result
Autonomous Goal-Driven Tool-Using Memory Self-Correcting
  • ◆
    Perception
    Reads web pages, files, databases, APIs as context.
  • ◆
    Planning
    Breaks goal into sub-tasks with Chain-of-Thought reasoning.
  • ◆
    Action
    Calls tools: code interpreter, browser, API, terminal.
  • ◆
    Reflection
    Checks output quality, retries on failure autonomously.
  • ◆
    Examples
    AutoGPT, Devin, OpenAI Operator, Claude Computer Use.

Protocol Layer

MCP — Model Context Protocol

MCP is an open standard by Anthropic that lets AI agents securely connect to external data sources and tools — like a USB port for AI. Agents can read, write, and manipulate files, databases, APIs via MCP servers.

// MCP File Read & Write Example const mcpClient = new MCPClient({ server: 'file-system-mcp' }); // READ a file const content = await mcpClient .callTool('read_file', { path: '/data/report.txt' }); // WRITE updated content await mcpClient.callTool('write_file', { path: '/data/output.json', content: JSON.stringify(result) });
  • ◆
    Standardised Interface
    One protocol connects agents to any tool — no custom integrations.
  • ◆
    File System Access
    Agents read CSV, JSON, PDFs; write reports, configs, logs.
  • ◆
    Database & API
    Query Postgres, call GitHub API, push to Slack — all via MCP.
  • ◆
    Security
    Permission scopes ensure agents only access what they're authorised to.
